# Product-Analytics-using-API-SQL-Power-BI
🔹 Project Overview

This project demonstrates an end-to-end data analytics workflow by extracting product data from a REST API, storing it in SQL Server, and building an interactive Power BI dashboard for business insights.

🔹 Tech Stack

Python

REST API

SQL Server

Power BI

🔹 Workflow

Extracted product data from Fake Store API using Python.

Cleaned and transformed nested JSON data.

Loaded structured data into SQL Server.

Performed SQL-based KPI analysis.

Built interactive Power BI dashboard.

Designed project in modular GitHub structure.
